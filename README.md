# 服务器健康监测系统 - 使用说明

## 系统简介
这是一个现代化的服务器健康监测系统，可以实时监控多台服务器的CPU和内存使用情况，并提供可视化图表和告警功能。

## 🚀 快速启动
1. 确保已安装 Docker 和 Docker Compose
2. 在终端中运行：`./start.sh`
3. 打开浏览器访问：`http://localhost:5000`

## 📋 使用方法
1. **添加服务器**：在网页表单中输入服务器IP、用户名、密码
2. **查看实时监控**：系统会自动显示服务器的健康数据和图表
3. **查看告警**：系统会自动检测异常并显示告警信息
4. **查看历史数据**：点击服务器列表中的"历史"按钮
5. **删除服务器**：点击列表中的"删除"按钮

## 🎯 系统功能
- ✅ 多服务器实时监控
- ✅ 自动数据采集（通过SSH）
- ✅ 可视化图表展示
- ✅ 智能告警系统
- ✅ 历史数据查询
- ✅ 响应式网页设计

## 📁 项目结构
- `Dockerfile` - 容器配置
- `docker-compose.yml` - 服务编排
- `app.py` - 主程序
- `data/` - 数据存储目录
  - `hosts.json` - 服务器配置
  - `history_*.csv` - 历史监控数据

## 🔧 技术栈
- 后端：Python Flask
- 前端：HTML + JavaScript + ECharts
- 数据采集：Paramiko (SSH)
- 容器：Docker + Docker Compose
